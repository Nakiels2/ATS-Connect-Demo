What you will need to integrate Workday and Lever

For The Job Requisition Sync

You will need:

  • A Report in Workday of All Job Requisitions
  • An ISU Account
  • The report to be owned by the ISU account
  • A Lever SBX Environment.
  • API Key with all permissions assgined.
    • Go to Lever
    • Go to Settings
    • Go to API and Integrations
    • Click Create API Key
    • Create the Key

For the Hire Sync:

You will need:

  • Candidates Archived as Hired in Lever
  • The offer form configured in Lever
  • API Key with all permissions assgined.
